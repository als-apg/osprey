# Lattice Physics Example Scripts

Reference scripts for common AT (Accelerator Toolbox) workflows.
Claude Code reads these to learn patterns for generating lattice physics code.

## Scripts

| Script | Purpose |
|--------|---------|
| `load_and_inspect.py` | Load a lattice file and print element summary |
| `compute_optics.py` | Compute Twiss parameters and linear optics |
| `fit_tune.py` | Adjust quadrupole strengths to match target tunes |
| `fit_chromaticity.py` | Adjust sextupole strengths to correct chromaticity |
| `plot_optics_plotly.py` | Interactive Plotly optics plot saved as HTML artifact |
