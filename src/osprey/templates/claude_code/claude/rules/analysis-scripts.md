# Analysis Scripts

Write analysis scripts to `osprey-workspace/scripts/` using descriptive filenames
(e.g., `analyze_beam_current.py`, `compute_optics.py`, `plot_bpm_orbit.py`).
Scripts in this directory are automatically executed and results returned to you.

- The Write tool displays code as a clean diff — easier to review than inline MCP parameters
- Results include stdout, execution status, and any saved artifacts
